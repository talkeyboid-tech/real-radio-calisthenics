これは全国各地のラジオ体操動画の情報を取得する API です。

## 🛠️ Installation

```bash
npm install
```

## 🚀 Serving API

サーバ起動

```bash
npm run start
```

サーバ起動（ホットリロード）

```bash
npm run dev
```

## 👮‍♂️ Our Policy

私達は全国の硬派なラジオ体操を求めています。利用に際し以下の条件がありますのでご注意ください。

✅ 推奨事項

- 硬派なラジオ体操動画を投稿すること
- 軟派なラジオ体操を見つけたら管理者に報告すること

🆖 禁止事項

- 存在しない動画 ID を投稿すること
- 軟派なラジオ体操動画を投稿すること
